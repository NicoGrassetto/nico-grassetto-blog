# Data Visualization in Blog Posts

Welcome to this demo showing how to embed charts and graphs in your blog posts! This post demonstrates various types of charts that you can use to visualize data in your content.

## Line Chart Example

Here's a simple line chart showing website traffic over time:

<LineChart 
  title="Website Traffic Over Time"
  xLabel="Month"
  yLabel="Visitors"
  labels={["Jan", "Feb", "Mar", "Apr", "May", "Jun"]}
  datasets={[
    {
      label: "Unique Visitors",
      data: [1200, 1900, 3000, 5000, 2300, 3200]
    },
    {
      label: "Page Views", 
      data: [2400, 3800, 6000, 10000, 4600, 6400]
    }
  ]}
  width={600}
  height={400}
/>

## Bar Chart Example

This bar chart shows programming language popularity:

<Chart 
  type="bar"
  data={{
    labels: ['JavaScript', 'Python', 'Java', 'TypeScript', 'C#', 'PHP'],
    datasets: [{
      label: 'Developer Usage (%)',
      data: [67.8, 44.1, 35.4, 34.8, 27.9, 20.9],
      backgroundColor: [
        '#f7df1e',
        '#3776ab', 
        '#ed8b00',
        '#3178c6',
        '#512bd4',
        '#777bb4'
      ]
    }]
  }}
  options={{
    title: "Programming Language Popularity 2024",
    scales: {
      y: {
        beginAtZero: true,
        title: {
          display: true,
          text: 'Usage Percentage'
        }
      }
    }
  }}
  width={600}
  height={400}
/>

## How to Use Charts in Your Posts

To use these charts in your blog posts, you have several options:

### Option 1: Using MDX (Recommended)

1. Create your blog post with a `.mdx` extension instead of `.md`
2. Import the chart components at the top of your file:
   ```mdx
   import Chart from '../../components/Chart.astro';
   import LineChart from '../../components/LineChart.astro';
   import ScatterChart from '../../components/ScatterChart.astro';
   ```
3. Use the components directly in your content with props

### Option 2: Using the Blog Layout

If you prefer to keep using `.md` files, you can modify your blog layout to include chart data in the frontmatter and render charts based on that data.

### Available Chart Types

The `Chart` component supports all Chart.js chart types:
- `line` - Line charts
- `bar` - Bar charts  
- `pie` - Pie charts
- `doughnut` - Donut charts
- `scatter` - Scatter plots
- `bubble` - Bubble charts
- `polarArea` - Polar area charts
- `radar` - Radar charts

### Chart Component Props

The main `Chart` component accepts these props:

- `type`: Chart type (required)
- `data`: Chart.js data object (required)
- `options`: Chart.js options object (optional)
- `width`: Chart width in pixels (default: 400)
- `height`: Chart height in pixels (default: 300)
- `id`: Custom canvas ID (optional, auto-generated)

### LineChart Component Props

The `LineChart` component accepts:

- `title`: Chart title
- `xLabel`: X-axis label  
- `yLabel`: Y-axis label
- `labels`: Array of x-axis labels
- `datasets`: Array of dataset objects with `label`, `data`, and optional `color`
- `width` & `height`: Dimensions

### ScatterChart Component Props

The `ScatterChart` component accepts:

- `title`: Chart title
- `xLabel` & `yLabel`: Axis labels
- `data`: Array of data series, where each series is an array of [x, y] coordinate pairs
- `labels`: Array of series labels
- `colors`: Array of colors for each series
- `width` & `height`: Dimensions
